Daily Mailer
项目介绍 配置入口

Private Console

进入配置中心

这是静态网页内的轻量登录，只用于隐藏配置生成器。真正的邮箱密码仍然只放在 GitHub Secrets。

Config Studio

生成新的 config.json

修改地区、新闻类型、邮件标题和显示时间，然后下载 config.json 上传覆盖 GitHub 仓库根目录。

显示时间 读取中 Asia/Shanghai

Schedule

发送计划

这里的时间用于配置展示。真正的自动运行时间还需要同步修改 .github/workflows/daily-mail.yml 里的 cron。

Mail

邮件外观

SMTP 账号、授权码和收件人不写入 config.json，请继续放在 GitHub Secrets。

Briefing

内容选择

新闻类型

Public Contact

访客联系方式

这些信息会写入公开的 config.json，并显示在主页“联系我们”里。
