Workflow

清晨之前，简报已经在邮箱里。

抓取天气Open-Meteo 获取当日天气。
整理新闻Google News RSS 按栏目聚合。
生成邮件输出 Apple 风格日报排版。
自动投递GitHub Actions 定时发送。

Privacy

敏感信息不放在网页里。

SMTP 密码、Gmail 应用专用密码、收件邮箱等信息由 GitHub Secrets 保存。公开网页只展示系统说明和公开配置生成入口。

Control

配置入口放在二级页面。

只有知道页面账号密码的人，才能进入配置生成器并下载新的 config.json。
